# Amazon Sales Analytics Dashboard | Power BI

## Project Overview
This project presents an interactive Power BI dashboard built to analyze Amazon sales data using the **Amazon Sales 2025** dataset sourced from Kaggle. The dashboard provides insights into sales performance, order trends, cancellations, and customer behavior to demonstrate how data analytics can support business decision-making in an e-commerce environment.

The project is designed as a portfolio demonstration for Data Analyst and Business Intelligence roles.

---

## Business Objectives
- Monitor overall sales performance and order volume
- Track order completion and cancellation rates
- Identify top-performing products and categories
- Analyze sales trends over time
- Support data-driven decisions related to inventory, pricing, and marketing

---

## Dashboard Features
- KPI cards for:
  - Total Sales
  - Total Orders
  - Completed Orders
  - Cancellation Rate
- Monthly, yearly, and category-level sales trend analysis
- Product-wise and payment method performance insights
- Interactive slicers for date and category-based filtering
- Executive-level dashboard layout for quick decision-making

---

## Tools & Technologies Used
- **Power BI** – Data modeling, visualization, and dashboard creation  
- **DAX** – Calculated measures and KPIs  
- **SQL** – Data understanding and analytical logic  
- **Data Analysis & Visualization Principles**

---

## Dataset Information
- **Dataset Name:** Amazon Sales 2025  
- **Source:** Kaggle  
- **Dataset Type:** E-commerce / Sales Transactions  
- **Data Includes:**
  - Order ID  
  - Order Date  
  - Product Name & Category  
  - Sales Amount  
  - Order Status (Completed / Cancelled)  
  - Payment Method  
  - Customer Information  

*The dataset used in this project is publicly available on Kaggle and is used strictly for educational and portfolio purposes.*

---

## Data Preparation & Modeling
- Cleaned and transformed raw sales data using Power BI
- Handled missing values, duplicates, and inconsistent formats
- Created calculated columns and measures using DAX
- Designed a star-schema style data model for efficient reporting
- Implemented date hierarchies for time-based analysis

---

## Key DAX Measures
- Total Sales  
- Total Orders  
- Completed Orders  
- Cancelled Orders  
- Completion Rate (%)  
- Cancellation Rate (%)  

These measures enable performance tracking across time periods, product categories, and customer segments.

---

## Business Impact & Insights
Although this is a portfolio project, the analysis demonstrates how insights derived from the **Amazon Sales 2025** dataset can support real-world business decisions, including:
- Identifying high-performing products and categories for inventory optimization
- Detecting cancellation patterns to minimize revenue loss
- Understanding seasonal sales trends to improve forecasting
- Enabling executive-level reporting through consolidated KPIs
